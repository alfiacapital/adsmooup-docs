📘 AdsMooup API Documentation

Base URL: https://adsmooup-backend.vercel.app/api

🔐 Authentication

POST/auth/login

Authenticate user and return a JWT token (set as cookie).

FormData:
  • email – string
  • password – string

GET/auth/logout

Logout the user and clear the token cookie.

GET/auth/me

Get authenticated user.

👤 User Endpoints

GET/users

Returns all users.

POST/users

Create User

FormData:
  • name – string
  • email – string
  • password – string

PUT/users/:id

Update a user by ID.

DELETE/users/:id

Delete a user by ID.

🍽️ Restaurant Endpoints

POST/restaurants

Create a new restaurant.

FormData:
  • name – string
  • logo – file (image)

GET/restaurants

Get all restaurants.

GET/restaurants/:id

Get a specific restaurant by ID (with categories and menus).

PUT/restaurants/:id

Update restaurant info.

DELETE/restaurants/:id

Delete a restaurant.

POST/restaurants/:id/categories

Add a category to a restaurant.

FormData:
  • name – string
  • image – file (image)

POST/restaurants/:restaurantId/categories/:categoryId/menus

Add menu images to a category.

FormData:
  • images[] – multiple files

📢 Advertisement Endpoints

POST/ads

Create an ad linked to a restaurant and category.

FormData:
  • image – file (image)
  • restaurantId – string
  • categoryId – string

GET/ads

Get all ads.

GET/ads/:id

Get an ad by ID.

DELETE/ads/:id

Delete an ad by ID.
